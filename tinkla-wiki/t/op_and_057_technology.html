OpenPilot Unity Branch and HV Battery Service Plan

OpenPilot Unity Branch and HV Battery Service Plan from 057 Technology

Many people asked if OpenPilot, because it works with CAN messages, would interfere with the HV Battery Service Plan offered by 057 Techology. The HV Battery Service Plan website states that:

Third party CAN devices which transmit on the CAN bus are prohibited. The use of third party CAN devices which transmit on the CAN bus can void this Service Plan, at 057 Technology's sole discretion, even if the device's intended purpose has nothing to do with the High Voltage Battery Pack.

I am happy to announce that after talking to them it was confirmed that OpenPilot does not interfere with the PowerTrain CAN bus in any way that would void the warranty of the HV Battery Service Plan. This is only valid for the Unity branch and the hardware configured and sold by Tinkla. I can not provide the same guarantees if you run off moddified branches which migh send other messages or affect the CAN BUS in other ways.
