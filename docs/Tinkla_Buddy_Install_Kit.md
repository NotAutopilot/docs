The Tinkla Buddy Install Kit is comprised of:

- RK3328 based ECU with dual Gb Ethernet, 1Gb RAM
- high endurance 32Gb SD card - pre-imaged
- RJ45 barrel for bypass
- 2.4/5 GHz USB WiFi adapter
- Fakra to RJ45 adapters (you will need two 2-3 ft CAT5e/CAT6 cables)
- USB power cable (use with Tinkla OBD-C adapter)

The install kit is available on our store.
